# git_check_commit_size
Can be used to query the change size of the most recent git commits.  
可以用来查询最近几次git提交的变更体积大小。
![git_check_commit_size_cover_0.png](https://github.com/iZerat/resource/blob/main/git_check_commit_size_cover_0.png)  

Place the git_check_commit_sider.sh file in the git repository and double-click to run it.  
把git_check_commit_size.sh文件放到git仓库里双击运行即可。
![git_check_commit_size_cover_1.png](https://github.com/iZerat/resource/blob/main/git_check_commit_size_cover_1.png)
